 <h5>Kata Sandi WiFi?</h5>
 <ol>
<li> Pindah ke atas satu folder (ke folder root)
di ES File Explorer dan Anda akan melihat
direktori bernama data.</li>
<li> Buka data > misc > folder wifi
dan Anda akan menemukan file bernama
wpa_supplicant.conf.</li>
<li> Buka file dan pastikan Anda menggunakan
penampil Intent/HTML bawaan ES File Explorer
untuk tugas ini.</li>
<li> Sekarang dalam file ini Anda akan dapat melihat
SSID jaringan dan kata sandinya.</li>
<li> Sekarang catat nama jaringan
dan kata sandinya</li>
<li> Dengan cara ini Anda akan dapat melihat semua
kata sandi wifi yang tersimpan di perangkat Android.</li>
<li> Harap jangan edit apa pun di
wpa_supplicant.conf karena dapat mengubah
kerja perangkat Anda atau akan mempengaruhi
konektivitas wifi Anda.</li>
</ol>

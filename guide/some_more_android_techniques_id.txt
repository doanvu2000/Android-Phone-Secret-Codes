 <h5>Pasangkan dengan mudah dengan NFC (Near Field
Communication) dan lakukan pembayaran</h5>
Anda dapat mengetuk dua perangkat untuk memasangkan
dan berbagi data dengan Near Field
Communication atau NFC. Pembayaran tagihan
dan tiket Anda disederhanakan oleh
perangkat yang mendukung NFC. Google dapat
menggunakan NFC untuk melakukan pembayaran aman dengan
hanya mengetuk ponsel Anda ke
prosesor pembayaran yang mendukung NFC lainnya
 <h5>Instal aplikasi keyboard pembaca pikiran</h5>
 Kita semua tahu bahwa keyboard Apple
tetap dan tidak dapat disesuaikan. Android
kita mengatakan ya untuk kustomisasi tersebut.
Android memungkinkan Anda mendapatkan keyboard
yang sangat baik seperti Slide IT, Swift Key dan
Swype. Keyboard ini tidak hanya
menyarankan kata tetapi juga belajar dari
gaya mengetik Anda dan menawarkan berbagai
opsi. Mereka menjadi lebih pintar seiring
waktu. Saya tidak tahu apakah beberapa orang
benar-benar dapat membaca pikiran kita tetapi aplikasi
ini membaca pikiran Anda dan
menyarankan kata saat mengetik. Bukankah ini
sangat menakjubkan! Kabar baiknya adalah
aplikasi keyboard Swift untuk perangkat Android
sekarang gratis dan tidak diragukan lagi adalah
aplikasi paling canggih dalam hal
keyboard cerdas.

 <h5>Transfer file menggunakan berbagai opsi</h5>
ANDA dapat memiliki banyak kontrol atas data
di Android dibandingkan dengan
sistem operasi lainnya. Wi-Fi direct dan
USB host adalah opsi terbaik untuk
mentransfer file besar. jika Anda ingin
mentransfer file berukuran 1 GB, Anda dapat melakukannya
dalam beberapa menit menggunakan Wi-Fi direct. Jaringan
Wi-Fi tidak diperlukan, karena
ponsel terhubung langsung.
USB host adalah opsi bagus lainnya dengan
mana Anda dapat menyalin-tempel file. Anda
dapat mencolokkan flash drive USB apa pun ke
ponsel dan mengakses hal-hal dengan mudah.

 <h5>Akses ke file</h5>
Tidak seperti Apple, Android memungkinkan Anda
mengakses semua file. Dengan file explorer
(seperti ES files explore dan file Manager)
Anda dapat memiliki akses ke sistem file lengkap
perangkat, yang tidak dapat Anda lakukan
di ponsel Apple.

<h5> Gunakan ponsel Anda sebagai hard drive portabel</h5>
Sebagian besar ponsel Android menerima kartu MicroSD.
pembaca kartu ini semakin
murah. Anda sebenarnya dapat menggunakan
ponsel Anda sebagai hard drive portabel
dengan kabel USB dari ponsel Anda ke
komputer.

Komputer Anda akan membaca ponsel Android Anda
sebagai drive eksternal. Yang harus Anda
lakukan adalah mengetuk tombol Hubungkan penyimpanan USB
yang akan muncul di
layar Android Anda. perlakukan seperti drive
USB lain yang Anda colokkan ke PC Anda.

Muat file yang ingin Anda bawa
ke suatu tempat dari PC Anda, dan Anda akan
dapat mengaksesnya dari
ponsel Anda atau komputer orang lain asalkan
Anda memiliki kabel yang tepat.

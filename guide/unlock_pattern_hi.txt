<h5>कमांड प्रॉम्प्ट का उपयोग करके Android लॉक पैटर्न को कैसे अनलॉक करें</h5>
<ol>
<li> अपने Android फोन को अपने
कंप्यूटर से कनेक्ट करें।</li>
 <li>  कमांड प्रॉम्प्ट को
व्यवस्थापक के रूप में खोलें।</li>
 <li> अब कमांड प्रॉम्प्ट विंडो में
निम्नलिखित कोड को सावधानीपूर्वक बिल्कुल
वैसे ही टाइप करें और फिर enter दबाएं।
adb shell cd /data/data/
com.android.providers.settings/
databases SQLite3 settings.db
update system set value=0 where
name=lock_pattern_auto lock; update
system set value-0 where name=lock
Screen.locked out permanently; .quit</li>
<li>  अब आप नीचे दिखाए गए अनुसार कुछ स्क्रीन देखेंगे
और फिर आप अपने फोन को रीबूट कर सकते हैं
और अब जब यह फिर से शुरू होगा
तो किसी भी रैंडम लॉक पैटर्न का उपयोग करके
अपने फोन को अनलॉक करने का प्रयास करें और यह
बहुत अच्छी तरह से काम करेगा।</li>
<li>  यदि आपको कोई समस्या आती है तो वही
चरण दोहराएं लेकिन उपरोक्त कोड के बजाय
adb shell rm/data/system/
gesture.key का उपयोग करने का प्रयास करें और फिर enter दबाएं और
अब यह देखने के लिए अपने डिवाइस को रीबूट करने का प्रयास करें कि क्या यह
काम करता है।
आनंद लें।</li>
</ol>

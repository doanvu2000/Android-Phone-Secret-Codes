<h5>فتح قفل النمط المقفل
دون فقدان البيانات؟</h5>
 +تحميل (Aroma File Manager).
+بطاقة ذاكرة لجهاز أندرويد.
+ جهاز أندرويد الخاص بك مقفل إما بـ
قفل النمط أو قفل كلمة المرور.
 1. 0 بعد تحميل Aroma File Manager
ضعه في بطاقة الذاكرة الخاصة بك (بطاقة SD).
أدخل بطاقة الذاكرة في جهاز
أندرويد المقفل الخاص بك.
 2. 0 افتح وضع الاسترداد الأصلي عن طريق
إعادة تشغيل جهاز أندرويد الخاص بك و
الضغط على مفتاح التشغيل ومفتاح رفع الصوت
في وقت واحد.
قد يكون للهواتف المختلفة طرق مختلفة
لفتح وضع الاسترداد الأصلي
لذا إذا واجهت أي مشكلة ابحث عنها في جوجل.
 30 . بعد الدخول في وضع الاسترداد استخدم
أزرار رفع الصوت وخفض الصوت للتمرير
لأعلى ولأسفل.
للاختيار استخدم الزر الأوسط.
 4. 0 من هناك انقر على "Install Zip from
SD Card" وحدد المسار لتثبيت "Aroma
File Manager" من بطاقة SD.
 5. 0 بعد التثبيت، سيفتح في وضع
الاسترداد.
 60 . من Aroma File Manager
انتقل إلى الإعدادات >> اذهب إلى الأسفل
وانقر على "AutoMount all devices on
start" ثم اخرج.
 7. 0 كرر الخطوة 4 و 5 مرة أخرى.
 8. 0 بعد هذا سيتم فتح "Aroma File
Manager" مرة أخرى.
 9. 0 الآن انتقل إلى مجلد Data >>
مجلد System وابحث عن >> "gesture.key"
أو "password.key" لقفل النمط أو
قفل كلمة المرور على التوالي.
 10. 0 احذف هذا الملف ثم اخرج من
مدير ملفات aroma وأعد تشغيل جهاز
أندرويد الخاص بك.
بعد إعادة التشغيل ستلاحظ أن
كلمة المرور أو قفل النمط لم تتم إزالته.
لكن لا تقلق ارسم أي نمط و
ستفتح جهاز أندرويد الخاص بك ولكن
تذكر أو دوّن ذلك النمط.

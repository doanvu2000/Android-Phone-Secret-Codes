 <h5>Mot de passe WiFi?</h5>
 <ol>
<li> Remontez d'un dossier (vers le dossier racine)
dans ES File Explorer et vous verrez un
répertoire appelé data.</li>
<li> Allez dans data > misc > dossier wifi
et vous trouverez un fichier nommé
wpa_supplicant.conf.</li>
<li> Ouvrez le fichier et assurez-vous d'utiliser
la visionneuse Intent/HTML intégrée d'ES File Explorer
pour la tâche.</li>
<li> Maintenant dans ce fichier vous pourrez voir
le SSID du réseau et leurs mots de passe.</li>
<li> Maintenant notez le nom du réseau
et son mot de passe</li>
<li> De cette façon vous pourrez voir tous les
mots de passe wifi enregistrés sur l'appareil Android.</li>
<li> Veuillez ne rien modifier dans
wpa_supplicant.conf car cela peut altérer le
fonctionnement de votre appareil ou affectera votre
connectivité wifi.</li>
</ol>
